# Solar Panel Sun Tracking System for Efficient Renewable Energy Generation

## Project Description
This project automatically tracks the direction of sunlight using LDR sensors and a DC motor. The system rotates the solar panel towards the maximum light source to improve solar energy generation efficiency. It supports renewable energy optimization and can be used as a basic solution for renewable energy monitoring in microgrids.

## Components Used
- Solar Panel
- LDR Sensors (2)
- L298N Motor Driver
- DC Geared Motor
- 9V Battery
- Jumper Wires

## Objective
To improve solar panel efficiency by automatically tracking sunlight.

## Applications
- Solar Power Plants
- Renewable Energy Systems
- Smart Energy Management
